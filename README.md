# Python Electricity Bill Calculator

## Description

A Python-based mini project that calculates electricity bills according to unit consumption slabs.

## Features

* Customer Name Input
* Meter Number Input
* Electricity Unit Calculation
* Slab-Based Billing System
* Fixed Charge Calculation
* Consumption Category Detection
* Bill Summary Generation

## Billing Slabs

* 0 - 100 Units → ₹5 per unit
* 101 - 200 Units → ₹7 per unit
* Above 200 Units → ₹10 per unit

## Technologies Used

* Python

## How to Run

1. Open the project in VS Code.

2. Open the terminal.

3. Run:

   py electricity_bill_calculator.py

4. Enter customer details and units consumed.
